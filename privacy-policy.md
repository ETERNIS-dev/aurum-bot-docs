# Aurum Bot — Privacy Policy
**Last updated: April 24, 2026**

## 1. Introduction
Aurum ("the Bot") is a Discord bot providing economy, leveling, and server management features. This Privacy Policy explains what data we collect, how we use it, and your rights regarding that data. By adding Aurum to your server or using its commands, you agree to this policy.

---

## 2. Data We Collect

### 2.1 User Data
For each Discord user who interacts with the Bot, we store:
- Discord User ID
- XP and level progress (per server)
- Wallet and vault balances
- Cooldown timestamps (daily, work, steal, beg)
- Purchased shop items and active perks (XP Boost, Vault Insurance)
- Equipped and unlocked titles
- Vault upgrade status

### 2.2 Server (Guild) Data
For each server the Bot is added to, we store:
- Discord Server (Guild) ID
- Bot configuration settings (prefix, cooldowns, XP rates, multiplier)
- Level-up message and channel settings
- Bot channel, notification channel, and level-up channel IDs
- Level roles and level role titles
- No-XP channel and role lists
- Shop items and custom titles
- Rank card customization settings (Premium)
- Leaderboard settings (Premium)
- Premium status and expiry date

### 2.3 Premium Key Data
When a premium key is generated or redeemed, we store:
- The key string
- Whether it has been used
- The Guild ID it was redeemed on

### 2.4 Support Server Data
When a user joins our support server, we check whether they are a server owner of any mutual guild with the Bot. If they are, they may be automatically assigned a designated role. No additional data is stored for this process.

---

## 3. How We Use Your Data
All data collected is used exclusively to:
- Operate the Bot's economy and leveling systems
- Persist server configuration across restarts
- Manage premium subscriptions and key validation
- Send global announcements and premium notifications to designated channels
- Assign the server owner role in our support server

We do **not** sell, share, rent, or use your data for advertising, analytics beyond bot operation, or any purpose outside of the above.

---

## 4. Data Storage & Security
- All data is stored in a MongoDB Atlas database hosted in the cloud.
- Only the bot developer has direct access to the database.
- Data is stored securely and access is protected by authentication.
- Railway is used for hosting the Bot. Both MongoDB Atlas and Railway have their own security and privacy practices.

---

## 5. Data Retention
- User and server data is retained as long as the Bot remains in a server or a user continues to interact with it.
- You may request deletion of your personal data at any time by contacting the developer.
- Removing the Bot from your server does not automatically delete server data. Contact the developer to request a full deletion.

---

## 6. Global Announcements
The Bot may send global announcements from the developer to all servers it is in. These messages are sent to a server's designated notification channel (or fallback channel) and may ping server management roles. No personal user data is included in these announcements.

---

## 7. Third-Party Services
The Bot uses the following third-party services:
- **MongoDB Atlas** — database storage ([Privacy Policy](https://www.mongodb.com/legal/privacy-policy))
- **Railway** — bot hosting ([Privacy Policy](https://railway.app/legal/privacy))
- **Discord** — the platform the Bot operates on ([Privacy Policy](https://discord.com/privacy))

We are not responsible for the data practices of these third-party services.

---

## 8. Children's Privacy
The Bot is not directed at children under the age of 13. We do not knowingly collect data from children under 13. If you believe a child has provided us with personal data, please contact us for removal.

---

## 9. Your Rights
You have the right to:
- Request a copy of the data we hold about you
- Request correction of inaccurate data
- Request deletion of your data
- Withdraw consent at any time by removing the Bot from your server

To exercise any of these rights, contact the developer through our support server.

---

## 10. Changes to This Policy
We may update this Privacy Policy at any time. Continued use of the Bot after changes constitutes acceptance of the updated policy. Significant changes will be announced via the Bot's global announcement system.

---

## 11. Contact
For privacy requests or questions, contact the developer through the **Aurum Support Server**.
